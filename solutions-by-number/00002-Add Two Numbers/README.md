# 00002 Add Two Numbers

- Platform: LeetCode
- Difficulty: Medium
- Topics: Linked List, Math, Recursion
- Language: Python
- Date: 2026-01-04

## Approach
(내 풀이 흐름 요약 3~6줄. 정답 복붙이 아니라 “재현 가능한 이해”를 목표로 작성)

주어진 두 개의 숫자 링크드 리스트를 정수형 숫자로 변환한다
이후 변환한 숫자를 더한 뒤, 더하여 나온 결과를 다시 한 자리씩 링크드 리스트로 변환한다
변환한 결과를 리턴 한다


## Complexity
- Time: O(n^2 + m^2)
- Space: O(n + m + k)

## Edge Cases / Counterexamples
- [0], [0] -> [0] ✅

## Mistakes / Lessons
- 다 풀고 보니 시간 복잡도가 좀 높아 보이긴 한다
